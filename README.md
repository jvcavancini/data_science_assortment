# data_science_assortment
Projeto de Mestrado (UFRJ - Ciência da Computação) em Data Science. Implementa um sistema dinâmico de recomendação de produtos. Utiliza análise de Ganho Marginal (Pareto) para definir o $K$ ótimo de itens por categoria e clustering K-Means para segmentação de preço, garantindo alta performance e variedade.
